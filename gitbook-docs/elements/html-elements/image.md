# Image

You can browse any images from your Media Library or upload a new image. You can define the dimensions as well as the alignment and an optional image link.

You can define [conditional logic](../../features/advanced/conditional-logic.md) on images to display an image based on a user selection if needed.

The images are compatible with the [PDF Generator](../../features/integrations/pdf-generator.md) Add-on.
